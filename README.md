💪 Gym Membership System (Python CRUD Project)

A simple and beginner-friendly **Gym Membership Management System** built with simple Python function.  
This project demonstrates the core concepts of CRUD operations (Create, Read, Update, Delete).

It is designed as a capstone project to practice real-world programming logic and data handling.

---
📌 About the Project

This system helps manage basic gym operations such as:
- Managing gym members
- Handling membership plans
- Recording payment transactions

All data is stored using Python lists and dictionaries, making it lightweight and easy to run without any external database.

---

Key Features:
👤 Member Management
- Add new members
- View all registered members
- Update member information
- Delete members

📦 Membership Plans
- Create new gym plans
- View available plans
- Update plan details
- Remove plans

💳 Payment System
- Record new payments
- View payment history
- Update payment records
- Delete payment data
